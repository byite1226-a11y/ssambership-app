### 0-2. 빌드·설치 명령 (Flutter 3.44.4)

```bash
# 0) 의존성
flutter pub get

# 1) .env 준비 — .env.example 복사 후 운영 값 입력(운영 콘솔에서 발급)
#    SUPABASE_URL=https://<project-ref>.supabase.co
#    SUPABASE_ANON_KEY=<anon key>
cp .env.example .env   # 이후 값 채우기

# 2) 디버그 설치·실행(개별질문 작성 on — 첨부·필기 QA 는 이 모드로)
flutter run --dart-define=IQ_CREATE_ENABLED=true

# 3) 릴리즈 검증 빌드(스토어 트랙과 동일 스위치 조합 확인용)
flutter build apk --dart-define=IQ_CREATE_ENABLED=true          # QA용(작성 on)
flutter build apk                                               # 스토어 기본(작성 off) 스모크
# iOS: flutter build ipa --dart-define=IQ_CREATE_ENABLED=true (서명 설정 후)
```
